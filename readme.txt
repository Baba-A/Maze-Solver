To Compile and run this program: ( On OSX or Linux )

1. Open the Terminal application/comman Prompt.

2. Navigate into the maze directory.

3.Compile the program with the following command : g++ Maze.cpp MazeApp.cpp -o maze

  You can also compile the program using the makefile : make

4. Run the executable file with: ./maze
